EXFILTRATION FIELDS:
  log_id          - Unique log/victim identifier
  ip              - Victim's IP address
  geo             - Geolocation data
  build_name      - Malware campaign name
  build_version   - Malware variant version
  pers_password   - Password from fake dialog (if entered)
  passwords       - Stolen browser passwords
  wallets         - Crypto wallet data
  credits         - Stolen credit cards
  autofills       - Browser autofill data
  is_vm           - Virtual machine detection flag
  hardwares       - System hardware info
  force_mode      - Operational mode flag
  uid             - Unique victim ID

JSON KEYS (for HTTP POST):
  buildName, buildVersion, uid

  Content-Type: application/zip
  Protocol:     'UPLD connect'
